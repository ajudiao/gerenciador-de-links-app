## 📱 Sobre o projeto

**Gerenciador de Links** é um aplicativo mobile desenvolvido com **React Native** que permite salvar, categorizar e acessar seus links favoritos de forma simples e prática — sem depender de navegadores ou favoritos perdidos.

Ideal para guardar artigos, tutoriais, referências de projetos, redes sociais e qualquer URL que você queira ter sempre à mão.

---

## ✨ Funcionalidades

- 📌 **Salvar links** com título, URL e categoria
- 🗂️ **Organizar por categorias** personalizadas
- 🔍 **Buscar** links rapidamente pelo título
- 🗑️ **Remover** links que não precisar mais
- 📋 **Copiar URL** com um toque
- 🌐 **Abrir link** direto no navegador

## 🚀 Como rodar o projeto

### Pré-requisitos

- [Node.js](https://nodejs.org/) (v18 ou superior)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)
- Aplicativo **Expo Go** no celular (iOS ou Android)

### Instalação

```bash
# Clone o repositório
git clone https://github.com/ajudiao/gerenciador-de-links-app.git

# Acesse a pasta do projeto
cd gerenciador-de-links-app

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npx expo start
```

### Rodando no dispositivo

1. Abra o **Expo Go** no seu celular
2. Escaneie o QR Code exibido no terminal
3. O app será carregado automaticamente 🎉
